# Notification System Design

## Overview
This repository contains a notification system design and implementation structure.

## Components
- `logging_middleware/` - middleware for logging requests and events.
- `notification_app_be/` - backend application for notification processing.
- `notification_app_fe/` - frontend application for notification UI.

## Notes
- Add backend code and API routes to `notification_app_be/`.
- Add frontend code and UI components to `notification_app_fe/`.
- Use `.gitignore` to ignore `node_modules/` and local environment files.
